# Demo Script

## Scenario 1: Ingest a document

Upload a PDF or Markdown file to the RAG API.

## Scenario 2: Ask a sourced question

Ask a question related to the uploaded document. The answer must include document and chunk references.

## Scenario 3: Prove sovereignty

Show that the platform uses Ollama inside the cluster and does not call external LLM APIs.

## Scenario 4: Prove operability

Open Grafana and show API latency, pod health, Longhorn volumes and logs.
